Шифровальщик текстов

Эта программа предназначена для шифрования и дешифрования текстов с использованием трех различных методов:

1. Шифр Цезаря
2. Шифр Виженера
3. Простая табличная перестановка


## Описание:

### Шифр Цезаря
Шифр Цезаря является одним из самых простых и известных методов шифрования. Он заключается в замене каждого символа входного текста символом, смещенным относительно его на некоторое фиксированное число позиций в алфавите. Например, при сдвиге на 3 символа вперед буква "А" заменится на "Г", "Б" на "Д" и так далее.

### Шифр Виженера
Шифр Виженера является методом полиалфавитного шифрования, который использует ключевое слово для шифрования. Каждая буква открытого текста заменяется на букву алфавита, смещенную на некоторое количество позиций, определяемое буквой ключевого слова на данной позиции. Таким образом, для каждой буквы открытого текста используется свой алфавит из 26 букв.

### Простая табличная перестановка
Простая табличная перестановка является методом перестановочного шифрования, в котором исходный текст записывается в таблицу по строкам, а затем считывается по столбцам. Размеры таблицы (количество строк и столбцов) определяются ключом шифрования.

## Использование
Программа имеет простой графический интерфейс, позволяющий выбрать метод шифрования/дешифрования, ввести исходный текст и ключ, а также получить результат обработки.

Для использования программы просто запустите исполняемый файл и следуйте инструкциям на экране.

## Требования
Программа написана на языке Python.

## Примечание
Эта программа предназначена исключительно для ознакомления с различными методами шифрования и не рекомендуется к использованию в целях обеспечения серьезной безопасности данных. Для серьезных задач шифрования следует использовать более современные и надежные алгоритмы шифрования.
